# Vertical arithmetic
```
Process two vectors by lane and store result to another vector.

Eg. C = A + B

       +-------+-------+-------+-------+
    A  |  a3   |  a2   |  a1   |  a0   |
       +-------+-------+-------+-------+
       +-------+-------+-------+-------+
 +  B  |  b3   |  b2   |  b1   |  b0   |
       +-------+-------+-------+-------+
 ---------------------------------------
       +-------+-------+-------+-------+
 =  C  | a3+b3 | a2+b2 | a1+b1 | a0+b0 |
       +-------+-------+-------+-------+
```


# Horizontal arithmetic
```
Reduce (aggregate) a vector to a scalar.

Eg. a = min(A), b = sum(B)

       +-------+-------+-------+-------+
    A  |  a3   |  a2   |  a1   |  a0   |  ==> a = min(a3, a2, a1, a0)
       +-------+-------+-------+-------+

       +-------+-------+-------+-------+
    B  |  b3   |  b2   |  b1   |  b0   |  ==> b = b3 + b2 + b1 + b0
       +-------+-------+-------+-------+
```
